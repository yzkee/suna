# Write Session Summary

Write a summary of this session to `workspace/.kortix/memory/`.

## Steps

1. Review what happened this session: tasks completed, decisions made, things learned, user feedback, unfinished work.

2. Write the summary to `workspace/.kortix/memory/sessions.md` (append to the file, most recent first):

```markdown
## [YYYY-MM-DD] Session Summary

**What happened:** [2-3 sentence overview]
**Completed:** [List of tasks done]
**Decisions:** [Key decisions with rationale]
**Learned:** [What was learned]
**Open items:** [Unfinished work]
```

3. Update `MEMORY.md` Scratchpad section: clear completed items, leave pending items for next session.

4. If anything from this session is worth remembering long-term, write it to the appropriate `memory/` file.

5. Confirm to the user what was captured.
