# Kortix

Self-host your AI Computer.

## Quick Start

Run the installer

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/kortix-ai/computer/main/scripts/get-kortix.sh)
```

The installer supports both:
- Local machine setup
- VPS/server setup (HTTPS + reverse proxy)
