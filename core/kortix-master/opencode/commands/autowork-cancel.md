# Autowork Cancel

<!-- KORTIX_AUTOWORK_CANCEL -->

**Autowork has been cancelled.**

Report to the user:

1. What was active — mode, iteration count, and current state
2. What is already done
3. What is still pending or blocked
4. Confirmation that continuation has stopped

Do **not** continue any pending work after this command.

To restart later, use `/autowork` for single-owner execution or `/autowork-team` for parallel execution.
